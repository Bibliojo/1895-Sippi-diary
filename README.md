
# Dr. Sippi Diary, 1895 — Digital Facsimile

This project presents a digital facsimile of the 1895 diary of Dr. Charles Sippi.  
The site provides a page-by-page viewing interface for high-resolution images of the original handwritten diary.

🔗 View the site: https://YOURUSERNAME.github.io/sippi-diary/

---

## About the Project

The diary is a handwritten record from 1895, offering insight into daily life, weather, social interactions, and local events.  

This digital version preserves the visual and material qualities of the original document while making it accessible for reading and research.

The project is part of a broader exploration of:
- transcription and interpretation of handwritten texts  
- text encoding (TEI)  
- digital presentation of historical materials
